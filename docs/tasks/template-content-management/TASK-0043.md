## 実装詳細

### 1. version catalog への追加 🔵

**信頼性**: 🔵 *architecture.md 技術的制約「datastore-preferences が既存依存関係に含まれているか確認が必要」・確認の結果未導入であることを確認済み*

`gradle/libs.versions.toml` に以下を追加する。

**実装ファイル**: `gradle/libs.versions.toml`

```toml
[versions]
# 既存に追加
datastore = "1.1.7"        # androidx.datastore-preferences の安定版（要バージョン確認）
# room は既存 = "2.7.1"（room-testing も同一バージョンを使用）

[libraries]
# 既存に追加
androidx-datastore-preferences = { group = "androidx.datastore", name = "datastore-preferences", version.ref = "datastore" }
room-testing = { group = "androidx.room", name = "room-testing", version.ref = "room" }
```

### 2. build.gradle.kts への追加 🔵

**信頼性**: 🔵 *既存 build.gradle.kts の dependencies ブロック構成より*

**実装ファイル**: `app/build.gradle.kts`

```kotlin
dependencies {
    // 既存に追加
    implementation(libs.androidx.datastore.preferences)
    // ...
    androidTestImplementation(libs.room.testing)
}
```

### 3. バージョン整合性の確認 🟡

**信頼性**: 🟡 *AGP 9.2.1 / compileSdk 36 環境との互換性確認が必要*

- datastore-preferences のバージョンは Compose BOM・compileSdk 36 と互換のある安定版を選択する
- 不整合がある場合はバージョンを調整する
