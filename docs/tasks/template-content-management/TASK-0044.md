## タスク概要

`Template` ドメインモデルから `vault` と `folder` フィールドを削除し、本文テンプレートを表す `body: String = ""` フィールドを追加する。テンプレートは「保存先」ではなく「ファイルの内容」のみを定義する責務に変更する。
