## タスク概要

TemplateEditScreen から vault/folder の入力エリアを削除し、本文テンプレートの複数行入力エリアを追加する。`{{content}}` を含む任意テキストを入力でき、特別なバリデーションは設けない。入力は `viewModel.updateBody` に接続する。

## 実装詳細

### 1. vault/folder UI の削除 🔵

**信頼性**: 🔵 *REQ-051・TC-051-01, TC-051-02 より*

**実装ファイル**: `app/src/main/java/com/den4dr/share2Obsidian/ui/template/TemplateEditScreen.kt`

- vault/folder の OutlinedTextField と関連ラベル・状態参照を削除する

### 2. 本文テンプレート入力エリアの追加 🔵

**信頼性**: 🔵 *REQ-052, REQ-053, NFR-003・dataflow.md フロー4 より*

```kotlin
OutlinedTextField(
    value = uiState.body,
    onValueChange = viewModel::updateBody,
    label = { Text(stringResource(R.string.template_label_body)) },
    minLines = 4,                       // 複数行・構造的テキスト対応 (NFR-003)
    modifier = Modifier.fillMaxWidth().testTag("template_body_field"),
)
```

- `{{content}}` を含む任意テキスト入力を許可（バリデーションなし、REQ-053）

### 3. 文字列リソース 🔵

**信頼性**: 🔵 *開発ルール「UI 文字列は strings.xml」より*

**実装ファイル**: `app/src/main/res/values/strings.xml`

- 本文テンプレートラベル・必要に応じてプレースホルダー説明文を定義する
