### 2. TopAppBar への設定アイコン追加 🔵

**信頼性**: 🔵 *REQ-002・architecture.md「EditScreen の変更点」より*

既存の `Scaffold` に `topBar` パラメータを追加する。
EditScreen が既に Scaffold を使用している場合は `topBar` スロットに追加し、
使用していない場合は `Scaffold` でラップする。

```kotlin
Scaffold(
    topBar = {
        TopAppBar(
            title = { /* 空またはアプリ名 */ },
            actions = {
                IconButton(onClick = onNavigateToSettings) {
                    Icon(
                        imageVector = Icons.Default.Settings,
                        contentDescription = stringResource(R.string.label_settings),
                    )
                }
            },
        )
    },
    bottomBar = { /* 既存のボタン */ },
) { paddingValues ->
    /* 既存のコンテンツ */
}
```

**注意**: EditScreen はすでに `Scaffold` を使用（bottomBar にキャンセル・送信ボタン）。
`topBar` スロットを追加するのみで OK。
