## 実装詳細

### 1. ShareContent sealed class 🔵

**信頼性**: 🔵 *interfaces.kt・REQ-001/101/201/301より*

**実装ファイル**: `app/src/main/java/com/den4dr/share2Obsidian/content/ShareContent.kt`

```kotlin
package com.den4dr.share2Obsidian.content

import android.net.Uri

sealed class ShareContent {
    data class Text(
        val text: String,
        val title: String? = null
    ) : ShareContent()

    data class Url(
        val url: String,
        val title: String? = null
    ) : ShareContent()

    data class Html(
        val html: String?,
        val fallbackText: String = "",
        val title: String? = null
    ) : ShareContent()

    data class File(
        val uri: Uri,
        val title: String? = null
    ) : ShareContent()
}
```
