Jsoup 1.18.3 を依存関係として追加し、INTERNET パーミッションを AndroidManifest.xml に追加する。既存の build 設定を壊さないように慎重に行う。

- [ ] `gradle/libs.versions.toml` に jsoup 1.18.3 のバージョン定義が追加されている
- [ ] `app/build.gradle.kts` の dependencies に `implementation(libs.jsoup)` が追加されている

## 実装詳細

### 1. libs.versions.toml へのJsoup追加 🔵

**信頼性**: 🔵 *architecture.md 依存関係追加セクションより*

**実装ファイル**: `gradle/libs.versions.toml`

```toml
[versions]
# 既存の versions に追記
jsoup = "1.18.3"

[libraries]
# 既存の libraries に追記
jsoup = { group = "org.jsoup", name = "jsoup", version.ref = "jsoup" }
```

### 2. app/build.gradle.kts への依存関係追加 🔵

**信頼性**: 🔵 *architecture.md 依存関係追加セクションより*

**実装ファイル**: `app/build.gradle.kts`

```kotlin
dependencies {
    // 既存の dependencies に追記
    implementation(libs.jsoup)
}
```

### 3. AndroidManifest.xml へのINTERNETパーミッション追加 🔵

**信頼性**: 🔵 *architecture.md・REQ-405より*

**実装ファイル**: `app/src/main/AndroidManifest.xml`

```xml
<manifest ...>
    <!-- 既存の manifest に追記 -->
    <uses-permission android:name="android.permission.INTERNET" />
    ...
</manifest>
```
