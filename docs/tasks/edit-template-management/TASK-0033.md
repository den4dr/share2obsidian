## 実装詳細

### 1. フィールド一覧 UI の追加

**信頼性**: 🔵 *REQ-031〜033より*

TASK-0032 で作成した TemplateEditScreen に以下を追加する。

- フィールド一覧セクション（「カスタムフィールド」ヘッダー付き）
- 各フィールド行: key名、valueSource ラベル、削除ボタン
- 「フィールドを追加」ボタン（または FAB）

**実装ファイル**: `app/src/main/java/com/den4dr/share2Obsidian/ui/template/TemplateEditScreen.kt`（TASK-0032 の続き）

---

### 2. フィールド追加/編集ダイアログ

**信頼性**: 🔵 *REQ-031〜033, REQ-041〜044より*

AlertDialog または BottomSheet として実装する。

- フィールドキー名入力 (OutlinedTextField)
- 値ソース選択 (RadioGroup または SegmentedButton): FIXED / HTML_META / URL / EMPTY
- FIXED の場合: デフォルト値入力フィールド
- HTML_META の場合: HtmlMetaKey ドロップダウン (OG_TITLE, OG_DESCRIPTION, URL, PUBLISHED_DATE, MODIFIED_DATE, AUTHOR)
- 値タイプ選択 (STRING / LIST)
- 「追加」ボタン

**実装ファイル**: `app/src/main/java/com/den4dr/share2Obsidian/ui/template/TemplateEditScreen.kt`

---

### 3. strings.xml への追加

**信頼性**: 🔵 *REQ-401より*

strings.xml に以下を追加する。

- `template_fields_header` - 「カスタムフィールド」
- `template_add_field_button` - 「フィールドを追加」
- `field_key_label` - フィールドキー名ラベル
- `field_value_source_label` - 値ソースラベル
- `field_value_type_label` - 値タイプラベル
- `field_source_fixed` - 「固定値」
- `field_source_html_meta` - 「HTMLメタ」
- `field_source_url` - 「URL」
- `field_source_empty` - 「なし」
- `field_type_string` - 「文字列」
- `field_type_list` - 「リスト」
- `html_meta_og_title` - 「OGタイトル」
- `html_meta_og_description` - 「OG説明」
- `html_meta_url` - 「URL」
- `html_meta_published_date` - 「公開日」
- `html_meta_modified_date` - 「更新日」
- `html_meta_author` - 「著者」

**実装ファイル**: `app/src/main/res/values/strings.xml`
