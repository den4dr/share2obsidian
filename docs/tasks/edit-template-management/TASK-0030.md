## 実装詳細

### 1. TemplateListScreen コンポーザブル

**信頼性**: 🔵 *architecture.md REQ-001, REQ-011より*

```kotlin
@Composable
fun TemplateListScreen(
    viewModel: TemplateListViewModel = hiltViewModel(),
    onNavigateBack: () -> Unit,
    onNavigateToEdit: (id: Long?) -> Unit,  // null=新規, Long=編集
)
```

- `uiState.collectAsState()` でテンプレートリストを取得
- TopAppBar に「テンプレート管理」タイトルと「←」戻るボタン
- LazyColumn でテンプレート一覧表示（各行: テンプレート名、デフォルトバッジ、削除ボタン）
- FloatingActionButton（+）で onNavigateToEdit(null) コール
- 削除確認ダイアログ（AlertDialog）

**実装ファイル**: `app/src/main/java/com/den4dr/share2Obsidian/ui/template/TemplateListScreen.kt`
