### 2. TemplateFieldEntity.kt

**信頼性**: 🔵 *database-schema.kt REQ-033, REQ-061より*

外部キー（CASCADE DELETE）とインデックスを持つ。`valueSource`、`valueType`、`metaKey` は Enum.name() を使い String として保存する。

```kotlin
@Entity(
    tableName = "template_fields",
    foreignKeys = [
        ForeignKey(
            entity = TemplateEntity::class,
            parentColumns = ["id"],
            childColumns = ["templateId"],
            onDelete = ForeignKey.CASCADE,
        )
    ],
    indices = [Index(value = ["templateId"])],
)
data class TemplateFieldEntity(
    @PrimaryKey(autoGenerate = true) val id: Long = 0,
    val templateId: Long,
    val key: String,
    val valueSource: String,   // FieldValueSource.name()
    val valueType: String,     // FieldValueType.name()
    val defaultValue: String,
    val metaKey: String?,      // HtmlMetaKey.name() または null
    val sortOrder: Int,
)
```

**実装ファイル**: `app/src/main/java/com/den4dr/share2Obsidian/data/db/TemplateFieldEntity.kt`

### 4. TemplateDao.kt

**信頼性**: 🔵 *database-schema.kt REQ-011〜014, REQ-021〜022, REQ-061より*

```kotlin
@Dao
interface TemplateDao {
    @Transaction
    @Query("SELECT * FROM templates ORDER BY isDefault DESC, id ASC")
    fun getAllTemplatesWithFields(): Flow<List<TemplateWithFields>>

    @Transaction
    @Query("SELECT * FROM templates WHERE isDefault = 1 LIMIT 1")
    suspend fun getDefaultTemplateWithFields(): TemplateWithFields?

    @Transaction
    @Query("SELECT * FROM templates WHERE id = :id")
    suspend fun getTemplateWithFieldsById(id: Long): TemplateWithFields?

    @Insert(onConflict = OnConflictStrategy.REPLACE)
    suspend fun insertTemplate(template: TemplateEntity): Long

    @Insert(onConflict = OnConflictStrategy.REPLACE)
    suspend fun insertFields(fields: List<TemplateFieldEntity>)

    @Delete
    suspend fun deleteTemplate(template: TemplateEntity)

    @Query("DELETE FROM template_fields WHERE templateId = :templateId")
    suspend fun deleteFieldsByTemplateId(templateId: Long)

    @Query("UPDATE templates SET isDefault = 0 WHERE id != :excludeId")
    suspend fun clearDefaultExcept(excludeId: Long)

    @Query("UPDATE templates SET isDefault = 0")
    suspend fun clearAllDefaults()
}
```

**実装ファイル**: `app/src/main/java/com/den4dr/share2Obsidian/data/db/TemplateDao.kt`

## 注意事項

- `valueSource`、`valueType`、`metaKey` は String として保存（`Enum.name()` を使用）
- Room テストには `room-testing` + `androidx.test.ext:junit` が必要（既存のテスト依存を確認）
- `exportSchema = true` の場合、`app/schemas/` ディレクトリが必要（KSP が自動生成）
- `@Transaction` は `getAllTemplatesWithFields()` など Relation を含むクエリに必須
- Gradleコマンドは必ず `mise exec -- ./gradlew ...` で実行すること
