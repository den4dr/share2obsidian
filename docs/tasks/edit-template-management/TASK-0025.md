## 実装詳細

### 1. gradle/libs.versions.toml への追加

**信頼性**: 🔵 *architecture.md Hilt セットアップより*

バージョン定義セクションに以下を追加する。

```toml
[versions]
room = "2.7.1"
hilt = "2.56"
ksp = "2.2.10-1.0.25"
hiltNavigationCompose = "1.2.0"

[libraries]
room-runtime = { group = "androidx.room", name = "room-runtime", version.ref = "room" }
room-ktx = { group = "androidx.room", name = "room-ktx", version.ref = "room" }
room-compiler = { group = "androidx.room", name = "room-compiler", version.ref = "room" }
hilt-android = { group = "com.google.dagger", name = "hilt-android", version.ref = "hilt" }
hilt-compiler = { group = "com.google.dagger", name = "hilt-compiler", version.ref = "hilt" }
hilt-navigation-compose = { group = "androidx.hilt", name = "hilt-navigation-compose", version.ref = "hiltNavigationCompose" }

[plugins]
hilt = { id = "com.google.dagger.hilt.android", version.ref = "hilt" }
ksp = { id = "com.google.devtools.ksp", version.ref = "ksp" }
```

**実装ファイル**: `gradle/libs.versions.toml`

---

### 2. プロジェクトルート build.gradle.kts への追加

**信頼性**: 🔵 *architecture.md Hilt セットアップより*

`plugins {}` ブロックに以下を追加する。

```kotlin
plugins {
    id("com.google.dagger.hilt.android") version "2.56" apply false
    id("com.google.devtools.ksp") version "2.2.10-1.0.25" apply false
}
```

**実装ファイル**: `build.gradle.kts`

---

### 3. app/build.gradle.kts への追加

**信頼性**: 🔵 *architecture.md Hilt セットアップより*

`plugins {}` ブロックに以下を追加する。

```kotlin
plugins {
    id("com.google.dagger.hilt.android")
    id("com.google.devtools.ksp")
}
```

`dependencies {}` ブロックに以下を追加する。

```kotlin
dependencies {
    implementation(libs.hilt.android)
    ksp(libs.hilt.compiler)
    implementation(libs.room.runtime)
    implementation(libs.room.ktx)
    ksp(libs.room.compiler)
    implementation(libs.hilt.navigation.compose)
}
```

**実装ファイル**: `app/build.gradle.kts`

---

### 4. AndroidManifest.xml への追加

**信頼性**: 🔵 *architecture.md Hilt セットアップより*

`<application>` タグに `android:name=".Share2ObsidianApp"` 属性を追加する。

```xml
<application
    android:name=".Share2ObsidianApp"
    ...>
```

**実装ファイル**: `app/src/main/AndroidManifest.xml`
