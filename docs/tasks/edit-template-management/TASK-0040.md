## 実装詳細

### 1. EditScreen.kt へのカスタムフィールドセクション追加

**信頼性**: 🔵 *architecture.md REQ-044, REQ-052より*

既存の EditScreen フォームの末尾（タグ/フォルダフィールドの下）にカスタムフィールドセクションを追加する。

表示条件: `formState.customFields.isNotEmpty()` の場合のみ表示する。

セクション構造:
```
[カスタムフィールド] (セクションヘッダー)

source        ← key ラベル（編集不可）
[https://example.com]  ← OutlinedTextField (編集可能)

description   ← key ラベル
[記事の概要]   ← OutlinedTextField (編集可能)
```

各フィールドの変更で `viewModel.updateCustomField(index, newValue)` を呼ぶ。

**実装ファイル**: `app/src/main/java/com/den4dr/share2Obsidian/ui/EditScreen.kt`（既存ファイルを変更）
