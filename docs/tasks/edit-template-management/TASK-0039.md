## タスク概要

`EditFormState` と `SendParams` に `customFields: List<CustomFieldState>` を追加し、`EditScreenViewModel` の `initialize()` メソッドが customFields を受け取り、`buildSendParams()` が customFields を含む SendParams を返すよう拡張する。

## 依存タスク

- **前提タスク**: TASK-0026, TASK-0038
- **後続タスク**: TASK-0040

- [ ] `EditFormState.kt` に `customFields: List<CustomFieldState>` が追加（デフォルト emptyList()）
- [ ] `SendParams.kt` に `customFields: List<CustomFieldState>` が追加（デフォルト emptyList()）
- [ ] `EditScreenViewModel.kt` に `initialize()` メソッドが customFields パラメータを受け取る
- [ ] `buildSendParams()` が customFields を含む SendParams を返す
- [ ] 単体テストが通る
