## タスク概要

`ProcessedContent` データクラスに `metadata: Map<HtmlMetaKey, String>` と `sourceUrl: String?` フィールドを追加し、`WebViewExtractionResult` に ogTitle, ogDescription, publishedTime, modifiedTime, author フィールドを追加する。いずれも既存コードとの後方互換を保つためデフォルト値を設定する。

## 依存タスク

- **前提タスク**: TASK-0025（HtmlMetaKey は TASK-0026 で定義）
- **後続タスク**: TASK-0036, TASK-0037

## 完了条件

- [ ] `ProcessedContent.kt` に `metadata: Map<HtmlMetaKey, String>` と `sourceUrl: String?` が追加
- [ ] `WebViewExtractionResult.kt` に ogTitle, ogDescription, publishedTime, modifiedTime, author フィールドが追加
- [ ] 既存のコンストラクタ呼び出しが後方互換（デフォルト値を提供）
- [ ] 単体テストが通る

## 実装詳細

### 1. ProcessedContent.kt の拡張

**信頼性**: 🔵 *interfaces.kt REQ-072より*

既存の `data class ProcessedContent(val body: String, val title: String? = null, val contentType: ContentKind)` に以下を追加する:

- `val metadata: Map<HtmlMetaKey, String> = emptyMap()`
- `val sourceUrl: String? = null`

既存コンストラクタ呼び出しはデフォルト値のため変更不要。

**実装ファイル**: `app/src/main/java/com/den4dr/share2Obsidian/content/ProcessedContent.kt`

---

### 2. WebViewExtractionResult.kt の拡張

**信頼性**: 🔵 *interfaces.kt REQ-071より*

既存フィールド (bodyText, pageTitle) に以下を追加する:

- `val ogTitle: String = ""`
- `val ogDescription: String = ""`
- `val publishedTime: String = ""`
- `val modifiedTime: String = ""`
- `val author: String = ""`

**実装ファイル**: `app/src/main/java/com/den4dr/share2Obsidian/util/WebViewExtractionResult.kt`
